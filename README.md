# MineSweeper
## 项目介绍
- 简单的扫雷游戏项目开发。
- 基于Java的SWING,AWT库设计的游戏。
- Linux应用开发结课时设计的项目，可以实现扫雷游戏基本的操作。
- 使用IDEA开发而来。
